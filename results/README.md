# results/ — per-run JSON outputs

Every invocation of `tracks/<tier>/<NN>_<name>/run.py` writes exactly one JSON
file into this directory. The file naming pattern is:

```
<tier>_<NN>_<name>__<solver>__seed<N>.json
```

For example:

```
core_01_foundation__torchgw-landmark__seed0.json
core_01_foundation__pot-entropic__seed0.json
```

The JSON schema is documented in `../CONVENTIONS.md` *(added in Task 2 of
the Phase 1 bootstrap plan)*. Files in this directory
are **not** committed to git (see `.gitignore`); they are regenerated by
running `scripts/run_tier.sh`. For release-time archival, copy the contents of
this directory into `../artifacts/v<X.Y.Z>/`.
